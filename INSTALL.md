# Installation Guide for CHRONO-ARCH

## Quick Install

```bash
git clone https://github.com/gitdeeper12/CHRONO-ARCH
cd CHRONO-ARCH
pip install -r requirements.txt
```

Requirements

· Python >= 3.9
· pip

Platform Support

· Linux ✅
· macOS ✅
· Windows ✅
· Termux ✅

Uninstall

```bash
pip uninstall chrono-arch
rm -rf CHRONO-ARCH
```
