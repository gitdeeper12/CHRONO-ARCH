# Contributing to CHRONO-ARCH

## How to Contribute

1. Report bugs via GitHub Issues
2. Suggest features with clear use cases
3. Submit pull requests

## Development

```bash
git clone https://github.com/gitdeeper12/CHRONO-ARCH
cd CHRONO-ARCH
pip install -e .[dev]
```

Code Style

· Black for formatting
· Type hints required
· Tests with pytest
